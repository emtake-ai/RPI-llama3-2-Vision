프로젝트명:
Edge Camera AI Control System

부제:
RTSP + YOLOv11 + Llama3.2:1B + RAG + LoRA 기반 분산 Edge Camera 관제 플랫폼

목표:

RTSP 기반 IP Camera와 Raspberry Pi 4B를 이용하여 Edge AI Camera Node를 구축한다.

각 Camera Node는 다음 기능을 수행한다.

* RTSP 영상 수신
* YOLOv11 Object Detection
* Detection Cache 생성
* Scene Summary 생성
* RAG 기반 지식 검색
* Llama3.2:1B 기반 자연어 응답
* HTTP REST API 제공
* 중앙 관제 연동
* Event 생성
* Alert 생성
* Health Monitoring

시스템은 xN개의 Camera Node로 확장 가능해야 한다.

중앙 관제 시스템은:

* 영상 확인
* 객체 확인
* Scene Summary 확인
* 자연어 질의
* Event 확인
* Alert 확인
* Camera 상태 확인
* Camera 설정 변경
* Camera 원격 관리

이 가능해야 한다.

추가 요구사항:

중앙 관제 시스템은 HTTP GET 및 HTTP POST를 이용하여 각 Camera Node와 자연어 질의를 수행할 수 있어야 한다.

예:

* "Camera 1 describe current scene"
* "Is there a person in Camera 3?"
* "Any dangerous situation in warehouse?"
* "What objects are detected?"
* "Summarize current activity"

Camera Node는 다음 정보를 조합하여 응답한다.

* Latest Frame
* YOLO Detection Result
* Detection Count
* Detection Confidence
* Camera Location
* Sensor Data
* Retrieved RAG Context
* Previous Scene Summary
* User Question

그리고 Llama3.2:1B는 JSON 형식으로 응답한다.
